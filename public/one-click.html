3

Deploy to DNS

✓

Choose your DNS provider. We'll call its API server-side using a token you paste here. The token is never stored — it's used for a single request.

Cloudflare → Use a scoped API token to add the TXT record automatically.
  1. Open Cloudflare → My Profile → API Tokens
  2. Create token → Template: Edit Zone DNS → select your zone → Continue → Create
  3. Paste the token below. It is sent once and discarded.
Required permission: Zone → DNS → Edit on the target zone.
Vercel → Only works if your domain uses Vercel DNS nameservers.
  1. Open Vercel → Account → Tokens
  2. Create a token (full account or team scope)
Netlify → Requires the domain to use Netlify DNS.
  1. Netlify → User settings → OAuth → Personal access tokens
  2. Generate a new token
Other providers (GoDaddy, Namecheap, Route53, Plesk, cPanel…)
Copy the TXT records below and paste them into your DNS control panel. Each provider has its own UI, but the values are the same.
