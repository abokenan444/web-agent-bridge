Try it yourself with cURL

curl -X POST https://webagentbridge.com/api/demo/execute \
     -H "Content-Type: application/json" \
     -d '{"action":"getPrice"}'

# Response: {"success":true,"price":"$49"}

curl -X POST https://webagentbridge.com/api/demo/execute \
     -H "Content-Type: application/json" \
     -d '{"action":"buy"}'

# Response: {"success":true,"message":"Product purchased successfully"}
