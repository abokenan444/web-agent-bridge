# WAB Browser Downloads

This directory stores the built browser installers for download.
Files placed here are served via the `/downloads/` route.

- `wab-browser-setup.exe` - Windows installer
- `wab-browser.dmg` - macOS disk image
- `wab-browser.AppImage` - Linux AppImage
- `wab-dns-provider.postman_collection.json` - Postman collection for Provider APIs
