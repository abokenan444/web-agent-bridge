Welcome to TechStore

This store is AI-agent-ready. It was integrated with WAB using just a script tag and 5 lines of code.

How this store was made WAB-enabled 
<!-- Step 1: Add the script -->
<script src="https://webagentbridge.com/script/wab.min.js"></script>

<!-- Step 2: Register your actions -->
<script>
  WAB.init({
    name: "TechStore",
    actions: {
      getPrice:  () => document.querySelector("#price").innerText,
      addToCart: ({ id }) => fetch("/wab/execute", { ... }),
      buy:       () => document.querySelector("#buy").click()
    }
  });
</script>
<!-- That's it. AI agents can now discover and use your store. -->
