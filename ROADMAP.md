# Web Agent Bridge (WAB) Roadmap 🗺️

Welcome to the WAB Roadmap! This document outlines where we are, what we are currently working on, and where we are heading. We believe in building in public and listening to our community.

## 🌟 Vision
To make every website on the internet natively understandable and interactive for AI agents, replacing fragile scraping with a robust, standardized protocol.

## ✅ Completed (v1.0 - v2.5)

### Core Protocol
- [x] Standardized `window.AICommands` interface
- [x] Auto-discovery of buttons, forms, and navigation
- [x] Schema.org JSON-LD extraction and action mapping
- [x] Granular permission system for agents

### Integrations & Ecosystem
- [x] **WordPress Plugin**: Native integration with shortcodes and hooks
- [x] **Cloudflare Worker**: Edge deployment support
- [x] **Shared Hosting**: Single PHP file endpoint (`wab-bridge.php`)
- [x] **Agent SDK**: Tools for Puppeteer/Playwright
- [x] **WAB-MCP Adapter**: Integration with Model Context Protocol

### Advanced Features (v2.x)
- [x] **Real-time Negotiation Engine**: Agents can negotiate prices
- [x] **Anti-Hallucination Shield**: Verifying DOM vs Vision
- [x] **Agent Workspace**: Premium UI for non-technical users
- [x] **Bilingual Support**: Full Arabic/English interface

---

## 🚧 Currently in Progress (v2.6)

- **Community Growth**: Building the foundation for open-source contributions (Discussions, Templates, Guides).
- **Enhanced Documentation**: Expanding the developer docs with more real-world examples.
- **Improved Error Handling**: Better self-healing selectors for dynamic SPAs (React/Vue/Angular).

---

## 📅 Planned (v3.0 & Beyond)

These are the major initiatives we are planning. The order and priority will be heavily influenced by community feedback.

### 1. Advanced Agent Mesh
- Fully decentralized peer-to-peer agent communication.
- Shared memory pools across different user sessions (opt-in).

### 2. Deeper E-commerce Integrations
- Native Shopify App.
- Magento/Adobe Commerce Extension.
- WooCommerce specific deep integrations (beyond standard WP plugin).

### 3. WAB Standard Proposal
- Drafting a formal specification to submit to web standard bodies (W3C/IETF).
- Creating a compliance test suite for website owners.

### 4. Enhanced Security & Privacy
- Zero-knowledge proofs for agent identity verification.
- Advanced stealth modes and anti-bot evasion techniques (for authorized agents only).
